Tentang Sistem

FRESH-ID

Sistem monitoring kesegaran makanan berbasis gas detection (NH₃, H₂S, CH₄) dengan teknologi sensor TGS2602, TGS2611, dan DHT22 untuk mendeteksi tingkat kesegaran sampel makanan secara real-time.

Sensor Gas
3 Tipe
TGS2602, TGS2611, DHT22
Sampel Makanan
4 Jenis
Ayam, Sapi, Ikan, Susu
Box Pengujian
12 Box
3 box per jenis sampel
Monitoring
Real-time
Update setiap 5 detik
