FRESH-ID

Smart Food Freshness Monitoring System

Deskripsi Sistem

FRESH-ID adalah sistem monitoring kesegaran produk makanan berbasis teknologi IoT yang menggunakan sensor gas (NH₃, H₂S, CH₄) dan sensor suhu/kelembaban (DHT22) untuk mendeteksi tingkat kesegaran produk secara real-time.

Fitur Utama

  • Real-time monitoring kesegaran produk
  • Multi-sensor detection (NH₃, H₂S, CH₄, Suhu, Kelembaban)
  • Alert system untuk kondisi abnormal
  • Data analytics dan visualisasi
  • Export data ke CSV/PDF
  • User management
  • Report generation
