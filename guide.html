Cara Memulai

  1. Buka file index.html di browser Anda
  2. Masukkan email Anda (contoh: user@email.com)
  3. Masukkan password (bebas untuk demo)
  4. Klik tombol "Masuk" untuk mengakses dashboard

Tip: Centang "Ingat saya" agar email Anda tersimpan

Menu Navigasi

  • Dashboard: Tampilan ringkasan nilai sensor terkini dengan status real-time
  • Monitoring: Grafik interaktif dengan data history dan fitur export
  • Tim Kami: Informasi lengkap anggota tim dan dosen pembimbing
  • Keluar: Logout dari sistem

Kustomisasi Website

1. Edit Informasi Tim

Buka file team.html dan ubah:

  • Nama anggota dan NIM (3 orang)
  • Nama dosen pembimbing dan NIP (2 orang)
  • Role/posisi masing-masing anggota

2. Membuat Logo

  • Buka file images/create-logo.html
  • Klik tombol "Download Logo PNG"
  • Simpan sebagai logo.png di folder images

3. Ubah Warna Theme

Edit file css/style.css bagian :root

Integrasi Hardware

Saat ini website menggunakan data simulasi (random)

Untuk menghubungkan dengan sensor real:

  • Setup backend server (Node.js, Python, dll)
  • Hubungkan sensor ke microcontroller (Arduino/ESP32)
  • Kirim data via HTTP REST API atau WebSocket
  • Edit fungsi generateSensorData() di file js/monitoring.js

Troubleshooting

Logo tidak muncul?

  • Pastikan file logo.svg ada di folder images
  • Atau generate logo PNG dari create-logo.html

Grafik tidak muncul?

  • Pastikan koneksi internet aktif (untuk Chart.js CDN)
  • Buka Console browser (F12) untuk cek error

Website tidak responsive?

  • Clear cache browser (Ctrl+Shift+Del)
  • Pastikan viewport meta tag ada di HTML

Siap Memulai?

Klik tombol di bawah untuk masuk ke halaman login
