Pendahuluan

FRESH-ID adalah sistem monitoring kesegaran makanan berbasis sensor gas yang mendeteksi kadar NH₃ (Amonia), H₂S (Hidrogen Sulfida), dan CH₄ (Metana) untuk menentukan tingkat kesegaran sampel makanan secara real-time.

Cara Login

  1. Akses halaman login sistem
  2. Masukkan email dan password yang telah terdaftar
  3. Klik tombol "Masuk" untuk mengakses dashboard

Dashboard

Dashboard menampilkan ringkasan data sensor aktif, jumlah pembacaan, status sampel segar, dan statistik lainnya dalam bentuk grafik dan kartu informasi.

Sensor & Pengujian

Sensor yang Digunakan

  • DHT22: Sensor suhu dan kelembaban udara
  • TGS2602: Sensor gas NH₃ (Amonia) dan H₂S (Hidrogen Sulfida)
  • TGS2611: Sensor gas CH₄ (Metana)

Sistem Box Pengujian

Setiap jenis sampel makanan (Ayam, Sapi, Ikan, Susu) memiliki 3 box pengujian yang masing-masing dilengkapi dengan sensor lengkap untuk monitoring kesegaran secara independen.

Interpretasi Hasil

  • Segar: NH₃ ≤ 25 ppm, H₂S ≤ 12 ppm, CH₄ ≤ 60 ppm
  • Kurang Segar: NH₃ 25-35 ppm, H₂S 12-20 ppm, CH₄ 60-80 ppm
  • Tidak Segar: NH₃ > 35 ppm, H₂S > 20 ppm, CH₄ > 80 ppm

Monitoring & Laporan

Data Sensor

Halaman Data Sensor menampilkan pembacaan real-time dari semua sensor dalam bentuk tabel dengan fitur pencarian, filter, dan export ke CSV.

Status Kesegaran

Halaman ini menampilkan overview status kesegaran dari semua box pengujian dengan indikator visual berwarna untuk memudahkan monitoring.

Laporan Pengujian

Sistem menyimpan riwayat semua pengujian yang dapat difilter berdasarkan tanggal, jenis sampel, dan status. Laporan dapat di-export dalam format PDF.

FAQ

Berapa lama waktu pengujian?

Pengujian dilakukan secara kontinyu dengan interval pembacaan setiap 5 detik (dapat dikonfigurasi).

Apa yang harus dilakukan jika sensor error?

Periksa koneksi sensor, restart sistem, atau hubungi tim teknis jika masalah berlanjut.

Bagaimana cara mengkalibrasi sensor?

Kalibrasi sensor dapat dilakukan melalui menu Pengaturan > Kalibrasi Sensor oleh admin.
