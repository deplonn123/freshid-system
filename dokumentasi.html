Tentang FRESH-ID

FRESH-ID adalah sistem monitoring kesegaran produk makanan berbasis sensor gas dan suhu. Sistem ini menggunakan teknologi IoT untuk memantau kondisi produk secara real-time.

Sensor yang Digunakan

  • TGS2602 - Sensor NH₃ (Ammonia) dan H₂S (Hydrogen Sulfide)
  • TGS2611 - Sensor CH₄ (Methane)
  • DHT22 - Sensor Suhu dan Kelembaban

Cara Menggunakan

  1. Login menggunakan akun yang terdaftar
  2. Pilih menu Dashboard untuk melihat overview semua box
  3. Gunakan menu Monitoring untuk live monitoring real-time
  4. Gunakan menu Data Sensor untuk melihat data historis
  5. Gunakan menu Status Kesegaran untuk melihat status per box
  6. Generate laporan melalui menu Laporan
