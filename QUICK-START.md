### Opsi 2: GitHub Pages (Lebih Profesional)

**Butuh install Git dulu**

1. **Install Git:**
   - Download: https://git-scm.com/download/win
   - Install dengan Next-Next-Finish
   - Restart komputer

2. **Buat akun GitHub:**
   - https://github.com/signup (gratis)

3. **Jalankan deploy.bat:**
   - Double-click file `deploy.bat` di folder ini
   - Ikuti instruksi di layar

4. **Website akan online di:**
   ```
   https://[username].github.io/[repo-name]/
   ```

Baca `DEPLOYMENT-GUIDE.md` untuk detail lengkap.
